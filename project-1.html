Project Overview: Assembly EndGame

The Assembly End Game is a final front-end challenge that brings together all key coding concepts into one project. It involves building a complete, functional app from scratch using components, routing, and real data. You’ll handle interactivity, layout, and responsiveness to create a polished user experience. The goal is to prove mastery by assembling everything learned into a single, production-ready web app.

Upon winning, the interface displays confetti and changes the button text to “New Game”, allowing a fresh start.
