Project 1: Tenzies

The app recreates the popular dice game Tenzies. The goal is simple but addictive — roll the dice until all ten show the same number. Players can “hold” dice between rolls to lock in values and strategize toward victory.

Live Link

Project Overview: Assembly EndGame

How it works: The app initializes with 10 dice, each showing a random number (1–6). Players click individual dice to “hold” them, preventing those dice from changing during future rolls. The Roll button re-rolls only unheld dice. When all dice are held and display the same number, the player wins.

Upon winning, the interface displays confetti and changes the button text to “New Game”, allowing a fresh start.
