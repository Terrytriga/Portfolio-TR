Project Overview: Tenzies

The Tenzies project from Scrimba is a fun, interactive dice game built with React that helps learners strengthen their understanding of component-based architecture, state management, and React hooks. The goal of the game is simple: roll ten dice until all of them show the same number. Players can “hold” dice to keep their values between rolls, while unheld dice change with each click of the roll button. Behind this playful mechanic, you learn to manage state effectively using useState, implement conditional rendering, and trigger side effects with useEffect—for example, to detect when the player has won. You also learn about mapping arrays to components, immutability in updating state, and structuring logic cleanly for UI interaction. Though small, Tenzies teaches the essential mental model of how React re-renders based on state changes, making it one of the best starter projects for solidifying your grasp of React fundamentals.

The Tenzies project is a fast-paced React game where you roll ten dice and try to make them all show the same number. You can “hold” dice to lock in values and keep rolling until you win, making it both simple and surprisingly addictive. Built with React, it highlights how state and component updates work together to create smooth, interactive experiences. It’s a perfect hands-on project to test your skills in building dynamic UIs and see React’s reactivity come to life in real time.
